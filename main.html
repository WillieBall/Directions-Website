Project Description:

For this project we are to make a multi-page/multi-tab web client using Html, JavaScript, bootstrap and CSS. The web pages are supposed to look good and be responsive, meaning that they collapse down into one column when shrunk down. This project is to use the "TomTom api" to pull directions to a certain location from a certain location with filters that are selectable for the type of route that the person would like. For each directions requests that is sent the system should store the returned JSON object in the database that we are using for this assignment (SQLite)
